# Changelog

All notable changes to this project are documented in this file.

## Unreleased

### Added

- LinuxServer.io-style reusable Docker image template.
- Reusable multi-architecture Docker build workflow.
- Secure `make`-based secret generation.
- Registry publishing design for GHCR, Docker Hub, GitLab, and Codeberg-compatible registries.
- Security scanning workflow with Trivy.
- Dependabot configuration for GitHub Actions.

### Changed

- PostgreSQL and MariaDB are maintained as standalone image repositories instead of examples in this template.
